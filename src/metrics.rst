Метрики
=======
 
Dice metric
-----------

``\[DM(S,M)=2\frac{|S\cap{}M|}{|S|+|M|}\]``

Jaccard index
-------------

``\[J(S,M)=\frac{|S\cap{}M|}{|S\cup{}M|}=\frac{DM(S,M)}{2-DM(S,M)}\]``
 
Hausdorff distance
------------------

``\[\mathrm{d}_{H}(A,B)=\max\left\{\sup_{a\in{}A}\inf_{b\in{}B}\mathrm{d}(a,b),\sup_{b\in{}B}\inf_{a\in{}A}\mathrm{d}(a,b)\right\}\]``
