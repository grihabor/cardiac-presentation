Метрики
=======
 
Dice metric
-----------

``\[DM(S,M)=2\frac{|S\cap{}M|}{|S|+|M|}\]``

Jaccard index
-------------

``\[J(S,M)=\frac{|S\cap{}M|}{|S\cup{}M|}=\frac{DM(S,M)}{2-DM(S,M)}\]``
 
Hausdorff distance
------------------

``\[\text{d}_{H}(A,B)  = \max\left\{ \sup_{a\in A} \inf_{b\in B} \text{d}(a,b),\sup_{b\in B} \inf_{a\in A}\text{d}(a,b)\right\}\]``
